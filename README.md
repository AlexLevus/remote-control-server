## 🏗️ Install & Run🏃
**Requirements:** Node.js 18.12.1

Download or copy repository:

```bash
git clone https://github.com/AlexLevus/remote-control-server.git
```

Install dependencies:
```bash
npm install
```

Run service:
```bash
npm run start:prod
```
